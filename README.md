# Magpie

Dotfiles of my minimal Unix system geared for media, programming & writing.

## Highlights

| Program   | Description          |
| --------- | -------------------- |
| zsh       | Shell                |
| neovim    | Text Editor          |
| lf        | File Explorer        |
| tmux      | Terminal Multiplexer |
| alacritty | Terminal Emulator    |
| bspwm     | Window Manager       |
| yabar     | Status Bar           |

## Dependencies

-  [Alfred](https://github.com/salman-abedin/alfred)
