# Magpie
Dotfiles geared for running a GNU/Linux system as minimally, efficiently and as fast as possible 

# Highlights
   * **zsh** (Shell)
   * **bspwm** (Window Manager)
   * **sxhkd** (Hotkey Daemon)
   * **alacritty** (Terminal Emulator)
   * **neovim** (Text Editor)
   * **lf** (File Manager)
   * **yabar** (Status Bar)
   * **rofi** (Dmenu Program)
